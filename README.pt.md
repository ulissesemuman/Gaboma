# Gaboma 🎲📖

Gaboma é um motor de **livro-jogo** (gamebook engine) desenvolvido em JavaScript, HTML e CSS.  
O objetivo é permitir que histórias interativas sejam criadas e jogadas diretamente no navegador, com escolhas que afetam o rumo da narrativa.

## 🚀 Como jogar

1. Acesse o jogo online pelo GitHub Pages:  
   👉 [Gaboma no GitHub Pages](https://ulissesemuman.github.io/Gaboma/)

2. O jogo será carregado no navegador e você poderá começar a explorar a história.

## 📂 Estrutura do projeto

- `books/` → Contém os capítulos e narrativas do livro-jogo.  
- `engine/` → Lógica principal do motor (interpretação das escolhas, fluxo da narrativa).  
- `language/` → Arquivos de idioma e textos para internacionalização.  
- `index.html` → Página inicial do jogo.  
- `loader.js` → Script de inicialização e carregamento.  
- `style.css` → Estilos visuais da interface.

## 🛠️ Como rodar localmente

Se quiser testar ou modificar o projeto no seu computador:

1. Clone o repositório:
   ```bash
   git clone https://github.com/ulissesemuman/Gaboma.git

2.  Abra o arquivo index.html em qualquer navegador moderno.

3. Edite os arquivos em books/ para criar novas histórias interativas.
