# Gaboma 🎲📖

Gaboma is a **gamebook engine** built with JavaScript, HTML, and CSS.  
It allows authors to create interactive stories where choices shape the narrative.

🌐 Available languages:  
- [English (default)](README.md)  
- [Português](README.pt.md)

## 🚀 How to Play

1. Play online via GitHub Pages:  
   👉 [Gaboma on GitHub Pages](https://ulissesemuman.github.io/Gaboma/)

2. The game will load in your browser and you can start exploring the story.

## 📂 Project Structure

- `books/` → Contains chapters and story content.  
- `engine/` → Core logic of the engine (choices, narrative flow).  
- `language/` → Language files for localization.  
- `index.html` → Entry point of the game.  
- `loader.js` → Initialization and resource loader.  
- `style.css` → Visual styles.

## 🛠️ Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ulissesemuman/Gaboma.git

2. Open the file index.html in any modern browser.

3. Edit files in books/ to create new interactive stories.
